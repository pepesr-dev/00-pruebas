clonar repositorio local

#Tercera prueba WorkFlow

1. Configuración inicial (Solo una vez)

    git init: Inicia un repositorio local en tu carpeta.

    git remote add origin <url-del-repo>: Vincula tu carpeta local con GitHub.

2. Ciclo diario (Subir cambios)

    git add .: Selecciona todos los archivos nuevos o editados.

    git commit -m "mensaje": Crea un "punto de guardado" con una descripción.

    git push origin main: Sube tus cambios a la rama principal en GitHub.

3. Gestión y estado

    git status: Te dice qué archivos has modificado y qué falta por subir.

    git log --oneline: Muestra el historial de cambios de forma resumida.

    git pull: Descarga y fusiona los cambios que otros hayan subido a GitHub.
